# MODS-Expanded
A MODS metadata plugin for Omeka with the full complement of categories as used by the Digital Commonwealth.

It extends the plugin made by <a href="http://www.columbia.edu/cu/libraries/inside/units/ldpd/">Columbia University Libraries Digital Library and Scholarly Technologies</a> to include the full scope of MODS fields used at Digital Commonwealth. Some of the structure and format have been retained so that the Omeka CMS will recognize it as a valid plugin.

## Installation

- All the `code` required to get started can be cloned from this repo.
- Clone the repo and upload the file to your Omeka install in the `HOME/applications/plugins/`folder.
- Enable the plugin in the admin interface of Omeka.
- It will display like this

<img src="https://digital.provath.org/uploads/MODSdisplay.png" title="Providence Athenæum" alt="Providence Athenæum" width="60%">
